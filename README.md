MBTI는 못 참지
======
MBTI 웹 애플리케이션입니다.

## Features

- create-react-app으로 간편한 번들링과 배포가 가능합니다.
- Spring Boot로 쉽게 서버 배포를 할 수 있습니다.

## Contributors

- FE: 김재현, 김준경, 안병민, 여주영
- BE: 박세련, 박철우, 이현수, 정순원
